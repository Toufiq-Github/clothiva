# **App Name**: Clothiva

## Core Features:

- Customer Authentication: Secure customer registration and login with email verification and password reset functionalities.
- Admin Authentication: Separate and secure admin login route with role-based access control (RBAC) to protect sensitive operations.
- Product Management: Comprehensive CRUD operations for products, including attributes like name, description, price, category, sizes, colors, stock, and images.
- Product Browsing: User-friendly product listing page with filters (category, price, size, color) and sorting options (price, new arrivals, keyword search).
- Shopping Cart: Persistent cart functionality allowing users to add, remove, and update items; seamless integration with checkout process.
- Checkout Flow: Streamlined checkout process including address selection/creation, order summary, payment method integration (e.g., Stripe/SSLCommerz), payment confirmation, and order success page.
- Order Management: Order tracking and history for customers, order management and status updates (Pending, Paid, Shipped, Delivered, Cancelled) for admins.

## Style Guidelines:

- Primary color: #EFEBE9 for a clean and modern look.
- Background color: #D7CCC8 provides a soft backdrop.
- Accent color: #8D6E63 and #A1887F for highlights and calls to action, creating a sense of sophistication.
- Headline font: 'Playfair' (serif) for elegant titles and headings; body font: 'PT Sans' (sans-serif) for readable and clean body text.
- Code font: 'Source Code Pro' (monospace) for any displayed code snippets.
- Use a consistent set of minimalist icons to represent different categories and functionalities.
- Employ a clean, grid-based layout to ensure readability and visual consistency.
- Subtle transitions and animations to enhance user experience, like cart updates and product image sliders.
